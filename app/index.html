About Me

Thanks for visiting my website. I am a Seattle-based Northwest native. I've worked in the software industry since 1997. I started out in the game industry and moved to back-end database development. Recently I levelled up my skills and added front-end.

I put this site together to showcase my skills. This page uses the Bootstrap grid and Font-awesome for the social media buttons. Please browse the portfolio for some more pages I've created.

Links to my social media are below. Please contact me if you'd like any more information.

LSEF

Website I created for Lake Stevens Education Foundation, a community service organization for which I was a volunteer board member.

LSEF provides scholarships, grants and programs for students and teachers in the Lake Stevens, WA school district.

The website, being the first one I created for other than personal use, is very simple. It uses just basic html and css and I wrote it entirely in Notepad, other than the calendar page which is a third-party widget.
